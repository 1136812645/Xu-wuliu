# 自测用例清单

## 1. 业务边界测试

| 编号 | 场景 | 输入 | 预期结果 |
| --- | --- | --- | --- |
| B01 | 零里程 | 里程=0 | 运单创建成功，干线运费=0，接口正常返回 |
| B02 | 零运费 | 阶梯运价命中 0 或里程=0 | 总费用正常计算，页面不报错 |
| B03 | 负数补贴 | subsidy=-20 | 总费用允许减少，费用明细展示负数补贴 |
| B04 | 正负金额混合 | 里程费+装卸费+保险费+补贴-扣款 | 各费用项独立展示，总金额精确到两位小数 |
| B05 | 空运单 | goodsName 为空白 | 接口拦截并返回明确错误信息 |
| B06 | 超重 | weightKg > maxWeightKg | 接口拦截，返回超出重量与建议拆分数 |
| B07 | 超体积 | volumeM3 > maxVolumeM3 | 接口拦截，返回超出体积与建议拆分数 |
| B08 | 同时超重超体积 | weightKg 和 volumeM3 均超限 | 双约束同时生效，系统不保存运单 |

## 2. 幂等测试

| 编号 | 场景 | 操作 | 预期结果 |
| --- | --- | --- | --- |
| I01 | 重复创建运单 | 相同 x-idempotency-key 连续提交两次 | 仅生成一张运单 |
| I02 | 重复签收 | 同一运单重复调用签收接口 | 仅首次生效，不产生脏数据 |
| I03 | 重复上传回单 | 同一运单重复调用 upload-pod | 仅首次生效，不重复更新数据库 |
| I04 | 未签收即上传回单 | 状态 ASSIGNED 时调用 upload-pod | 接口返回业务错误，不更新状态 |

## 3. 档案预警测试

| 编号 | 场景 | 输入 | 预期结果 |
| --- | --- | --- | --- |
| A01 | 驾驶证临期 | 到期日 30 天内 | 预警列表标记 EXPIRING_SOON |
| A02 | 驾驶证过期 | 到期日晚于今天 | 预警列表标记 EXPIRED |
| A03 | 车辆证件非法日期 | invalid-date 或空值 | 页面正常展示 INVALID，不抛异常 |
| A04 | 证件筛选 | 按过期状态查询 | 返回对应清单 |

## 4. MQ 异常测试

| 编号 | 场景 | 操作 | 预期结果 |
| --- | --- | --- | --- |
| M01 | 重复消息 | 同 eventId 消息重复投递 | 消费端幂等，不重复执行业务 |
| M02 | 非法消息体 | 缺少必要字段 | 进入死信队列，主流程不阻塞 |
| M03 | 队列不可用 | 关闭 RabbitMQ | 业务写入 outbox，待重试发布 |
| M04 | 消费异常 | 人为抛出异常 | 消息进入重试队列，超过阈值入 DLQ |

## 5. 并发与性能测试

| 编号 | 场景 | 操作 | 预期结果 |
| --- | --- | --- | --- |
| P01 | 50 并发开单 | 50 线程同时创建运单 | 成功请求数=运单数，无重复 |

## 6. 执行建议
