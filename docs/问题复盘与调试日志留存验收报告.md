- 重复签收请求虽然返回 idempotentBlocked=true, reason=ALREADY_SIGNED，
  但日志仍写入 waybill.signed，导致排查误判。

定位代码：
- 文件：apps/api/src/index.ts
- 路由：POST /api/waybills/:id/sign
- 当前实现：通过 shouldPublish 控制，只有真实状态变更才记录 waybill.signed。

- ALREADY_SIGNED / ALREADY_POD_UPLOADED 幂等拦截不再写“成功动作日志”。

修复文件：
- apps/api/src/index.ts
