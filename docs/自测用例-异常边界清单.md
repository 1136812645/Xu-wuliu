# 自测用例（异常与边界）

标识：自测用例
版本：v1.0
日期：2026-07-07

## 1. 适用范围

本清单覆盖以下验收范围：

- 业务边界：零里程、零运费、负数补贴、空运单、超限货物
- 幂等边界：重复签收、重复回单上传
- 并发边界：并发开单、并发修改同一车辆
- MQ 异常：消息丢失、重复消息、异常死信消息
- 档案边界：证件过期、证件空白、非法日期

## 2. 执行环境

- API 基地址：`http://127.0.0.1:3000`
- DB：`waybill_admin`
- MQ：RabbitMQ（可切换断连/恢复）
- 并发工具：JMeter / Postman Runner / k6（任选）

## 4. 判定标准

- P0 用例通过率必须为 100%。
- 不允许出现未捕获异常（500）替代业务错误码。
- 并发与 MQ 场景必须有数据库核对证据（数量、状态、去重记录）。
